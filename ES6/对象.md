# 对象
## 新增API
1. 成员速写
```
function sum(a, b, c){
    return {
        a,
        b,
        c
    }
}
此时 return 的a是 a=a;
```
---
2. 方法速写
<html>
对象字面初始化时，方法可以省略冒号和function关键字
</html> 
```
const obj = {
    a: 'name',
    age: 15,
    say () {
        console.log(this.a, this.age)
    }
}
```
---

3. 计算属性名
<html>
有的时候，初始化对象时，某些属性名可能来自于某个表达式的值.ES6中可以使用[name]表示这个名字是通过计算得到的。
</html>
```
const name = 'sayHello';
const user = {
    name: 'name',
    [name]() {
        console.log(this.name)
    }
}
user.sayHello();
```
