### 函数
##### 参数默认值(该函数自动变成严格模式)
```
function sum(a = 0, b = 1, c = 3){
    return a + b + c;
}
sum(10)
sum(1, null, null) 
传值必须传入的是undefined或者不传 才会取默认值
```
---
##### ...展开运算符
```
作为函数的参数使用
function sum(...args) {
    let sum = 0;
    for(let i = 0; i < args.lenght; i ++) {
        sum += args;
    }
    return sum;
}
> 对象展开
```
